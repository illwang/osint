# Proof of Timestamp / 预判时间戳存证

本目录用于保存 SSR 所有重大预判的**事前时间戳证据**。

## 存证原则

1. **事前记录**：所有预判必须在事实发生前完成 GitHub Commit 或 Release
2. **不可篡改**：使用 Git 哈希作为天然的时间戳证明
3. **偏差诚实**：验证失败的预判不删除，而是标注偏差原因并纳入下一轮 SEJ 校准

## 文件命名规范

```
YYYY-MM-DD_Front_[A/B/C]_Topic_Brief.md
```

示例：
- `2026-05-16_Front_B_Samsung_Strike_Countdown.md`
- `2026-05-17_Front_C_Transformer_Time_Premium.md`
