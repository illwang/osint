# OSINT 边界协议与自我审计纪律

> **版本**: 1.0.0  
> **日期**: 2026-05-17  
> **密级**: 公开方法论  
> **适用范围**: SSR 所有 L1/L2/L3 产品

---

## 一、核心原则：诚实边界即护城河

机构买家购买 OSINT，买的就是"基于证据的确定性"，而不是"聪明的猜测"。

任何为了追求"逻辑爽感"而跨越 OSINT 边界、滑向"内幕猜测"或"意图定性"的行为，都会损害 SSR 的信用货币。

---

## 二、三类常见越界行为与修正方案

### 越界类型一：意图推断（Intention Inference）

**典型错误**："宝钢蓄意增产绞杀定价"、"美方故意拖延谈判以争取时间"

**OSINT 边界**：企业战略意图、国家谈判策略属于**非公开决策过程**，除非有泄露的董事会纪要、外交电报或内部文件，否则 OSINT 无法验证。

**修正方案（客观战略后果审计）**：
- 删除一切"蓄意"、"故意"、"旨在"等意图词汇
- 替换为可观测的物理指纹：
  - **产能释放（Capacity）** → 环评公示、设备采购公告、招工信息
  - **价格压制（Price suppression）** → 国内现货价 vs 历史均值 vs 出口报价
  - **份额收割（Share capture）** → 海关出口量增速、海外中标公告频次

**示例修正**：
> ❌ 错误："宝钢蓄意增产压低 GOES 价格，绞杀欧美上游"
> ✅ 正确："宝钢 22 万吨新产能释放后，国内 GOES 现货价维持低位震荡，而出口变压器成品单价同比上涨 70%–100%，形成原材料内循环压价与产成品全球溢价的客观剪刀差"

---

### 越界类型二：非公开财务细节（Non-public Financial Granularity）

**典型错误**："西电合同负债中非美元结算占比提升"、"特变电工北美订单毛利率达 35%"

**OSINT 边界**：上市公司财报**不披露**合同负债的结算币种结构、不披露单一订单毛利率、不披露客户名称（除非达到披露阈值）。这些属于审计师工作底稿级别的信息。

**修正方案（代理变量审计）**：
- 用**可公开获取的财务比率**替代不可触及的明细：
  - 海外营收占比增速 vs 总收入增速
  - 合同负债绝对值增速 vs 营收增速（若预收款比例高，合同负债增速应显著快于收入增速）
  - 海外分部毛利率 vs 国内分部毛利率（年报分部报告）
  - 公开中标公告中的"联合体"结构（可推断风险分担比例）

**示例修正**：
> ❌ 错误："西电合同负债中非美元结算占比提升至 40%，主权安全性极高"
> ✅ 正确："西电海外营收占比从 2023 年的 12% 提升至 2025 年的 18%，同期合同负债增速（+45%）显著快于营收增速（+28%），推断海外客户为锁定交付时间支付了更高比例的预付款"

---

### 越界类型三：内幕级名单与精确排产（Insider Lists & Production Schedules）

**典型错误**："北美 5 家最高风险烂尾数据中心名单"、"SK 海力士第 3 季度 HBM 精确产能分配表"

**OSINT 边界**：特定企业的内部排产计划、客户的精确采购清单、未公开的土地交易细节，超出 OSINT 可验证能力。

**修正方案（筛选矩阵/方法论授权）**：
- 不卖"名单"，卖**"筛选方法论"**：
  - 维度一：公开报道的延期/取消信号（Bloomberg/Sightline Climate）
  - 维度二：地方公用事业公司（Utility）公开的**电力互联排队名单（Interconnection Queues）**变动
  - 维度三：项目地块在 Sentinel-2 卫星图像上的土建活跃度时间序列
- 交付物是**"三位一体验证算法"**，由机构客户的初级分析师自行跑数据
- 变现逻辑从"卖消息"升级为"**方法论授权费**"

## 三、L3 产品的"去内幕化"重构

### 原 L3 定位（危险）

> "包含北美 5 家烂尾数据中心名单、具体关税转嫁协议证据"

### 新 L3 定位（合规且可扩展）

> "包含完整参数边界、量化证伪阈值、对冲校准逻辑、以及可执行的筛选矩阵"

**L3 核心交付物清单**：

| 模块 | 内容 | 验证方式 |
|------|------|---------|
| **物理熔断窗口锚定** | 基于库存周转模型的全球炼厂库存硬约束日期 | 公开库存数据 + 物流 AIS |
| **立法 D-Day 投影** | 基于国会程序模型的 MATCH Act / HR 8283 全院表决时间窗 | Congress.gov + 委员会日程 |
| **资本-营收剪刀差切换阈值** | 财务触发器矩阵（需求验证成功/滞后/失败） | 财报数据 + 指引公告 |
| **CPS 传导系数** | 产能-价格-份额弹性映射表 | 海关统计 + 价格指数 |
| **时间溢价量化模型** | 二手设备转售溢价阈值、在手订单期权定价 | 二手平台爬虫 + 财报合同负债 |
| **卫星验证协议** | 土建活跃度评分标准（Sentinel-2 NDVI 变化率） | 公开遥感数据 |

## 四、能源成本的代理变量修正（案例）

**原越界表述**："中国享受伊朗的'1 级友好'低价油，中国变压器企业电力成本基数远低于欧美"

**修正后（可交易价差信号）**：

**监控指标**：
1. **SC-Brent 贴水率**：上海原油期货（SC） vs 布伦特（Brent）的月度价差均值
2. **中国原油进口来源集中度**：海关总署月度原油进口国别数据（中东/俄罗斯/非洲/美洲占比）
3. **中国制造业电力成本指数**：国家统计局工业生产者购进价格指数（电力子项）

**判定逻辑**：
- 若 SC-Brent 贴水在危机期间持续扩大，且中东进口占比提升 → 中国高耗能制造业享有**相对能源成本优势**
- 若中国工业电力购进价格指数涨幅显著低于欧美 PPI 能源子项 → 成本剪刀差确认

**价值点**：这是一个**可交易的价差信号**，比政治传闻更受交易员欢迎。

## 五、第一年"攒人品"路径（信用抵押物积累）

逻辑再硬，没有"战绩"也无法变现。信用需要可审计的时间戳。

### Phase 1：事件验证（0–3 个月）

- **5 月 18 日（周一）**：市场开盘，若 SC 原油和电力基建板块出现预判中的非对称走势
  - 立即在 GitHub 发 README 更新：《05/18 逻辑验证对账单》
  - 附上 5/17 提交的时间戳链接（`git log` 或 Release 页面）
- **5 月 20 日（下周三）**：霍尔木兹海峡通航量验证
  - 若通航量依旧低迷，再次更新对账单
- **发布策略**：不发邮件、不私信推销
  - 在雪球相关个股评论区（西电、特变电工等）只留一句话：
    > "5/17 预判已兑现，逻辑对账单见 GitHub：illwang/osint"
  - 目的：吸引同样盯着这些指标、且因看不准而痛苦的专业研究员**主动来找你**

### Phase 2：引用网络（3–6 个月）

- 向 3–5 家券商电新组实习生/助理分析师发送**免费观察**（非销售）
  - 主题："基于公开数据的变压器交付周期观察，供参考"
  - 内容：只给 L2 框架 + 验证日历，不给 L3 参数
  - 要求：若引用请标注来源 + GitHub 链接
- 目标：建立**"被引用记录"**，这是比任何自我宣传都硬的信用

### Phase 3：被动接单（6–12 个月）

- 当 GitHub Star > 100、雪球粉丝 > 1 万、有 3 次以上公开验证记录
- 机构研究员会开始主动搜索到你
- 此时价格不是你定的，是他们为了留住你而求你开的

## 六、审计声明

本协议由 SSR 方法论委员会制定，适用于所有公开发布与私人定制产品。

任何 SSR 分析师在输出结论前，必须完成以下自检清单：

- [ ] 本结论是否包含任何"意图"、"蓄意"、"旨在"等不可验证词汇？
- [ ] 本结论是否依赖非公开财务明细（结算币种、单一订单毛利率、精确排产）？
- [ ] 本结论是否包含无法通过公开渠道复核的"名单"、"精确数字"？
- [ ] 是否已为每个方向性判断附加了 3 个证伪条件？
- [ ] 是否已诚实标注不可知维度？
- [ ] 是否已在 GitHub 提交时间戳备份？

**通过自检的结论，才是 SSR 的信用货币。**
