## 1. Hormuz Escalation Index (HEI)

HEI 是加权复合指数，用于区分"噪音"（外交辞令）与"信号"（物理与市场行动）。

### 1.1 三因子架构

HEI 由三个核心维度构成：

| 分项 | 全称 | 定义 |
|------|------|------|
| **F** | 物理冲突烈度 (Force) | 交火频率、中立国船只遇袭事件、军事部署密度、硬杀伤实证 |
| **M** | 市场恐慌度 (Market) | 战争险保费曲线、原油期货波动率、库存异动、航运中断物理证据 |
| **D** | 外交进程置信度 (Diplomacy) | 双边/多边会谈成果、联合声明质量、协议执行可验证性 |

&gt; **L1 公开边界**：三因子的概念定义与校准逻辑公开。具体权重分配、实时估值与敏感性参数属于 L2/L3 审计内容。

### 1.2 极端事件声明（Critical Event Override）

**HEI 线性加权模型在离散型极端事件下会系统性低估风险。**

此类事件发生时，即使 HEI 数值未突破阈值，也应直接激活最高等级响应协议：

| 极端事件类型 | 响应级别 |
|-------------|----------|
| 证实误击巨型油轮并致沉没 | 最高响应 |
| 以色列因核红线触发单方面打击 | 最高响应 |
| 美军"铁锤行动"正式启动 | 一级响应 |
| 中国油轮发生第二起遇袭 | 一级响应 |
| 巴基斯坦因经济崩溃放弃调停 | 最高响应 |

&gt; **L1 公开边界**：触发事件类型公开。具体的 ΔHEI 预估变化与响应阈值属于 L2 审计内容。

## 2. Execution Entropy (EE)

**执行熵**是 SSR 的专有度量，用于衡量"政治声明"与"战术执行"之间的结构性落差。

### 2.1 概念定义

在高 EE 环境（如当前霍尔木兹危机）中，高层外交姿态（如伊朗外交部"两周方案"）会被底层战术升级（如 IRGC 扣押船只）**系统性抵消**。

**操作规则**：当 EE 处于高位时，所有外交"突破"均被视为**伪收敛（Pseudo-Convergence）**，直至物理证据（如排雷实证）确认信号。

### 2.2 当前评估维度（示例）

| 维度 | 声明层（外交辞令） | 执行层（物理事实） | EE 落差 |
|------|-------------------|-------------------|---------|
| 中美会谈 | "建设性战略稳定关系"、海峡保持开放原则共识 | 未发表联合声明；无具体执行机制；无排雷/监督/路线图 | **高** |
| 特朗普核谈 | 可接受"暂停核计划20年"（隐蔽松动） | 已拒绝伊朗最新方案；"耐心正在耗尽" | **极高** |
| 海峡控制权 | 特朗普宣称"我们控制着海峡" | 通行规则制度化仍在伊朗主导下推进 | **极高** |
| 停火状态 | 美防长称停火协议"依然有效" | 5/7多次交火；和谈陷入停滞；特朗普"不赞成停火" | **高** |

## 3. Gray Zone Institutionalization (GZI)

度量海上安全从"公共品"向"私有/条件化资产"的转移。

### 3.1 三因子架构

| 分项 | 全称 | 定义 |
|------|------|------|
| **B** | 双边化度 (Bilateralization) | 从 UNCLOS 标准转向逐船外交交易 |
| **P** | 私营化度 (Privatization) | 对 PMSC（私营军事安保）与非标准保险的依赖上升 |
| **C** | 条件化度 (Conditionality) | 区域强国实施的"友好等级"分层租金体系 |

### 3.2 刻度解读

| GZI区间 | 状态 | 含义 |
|---------|------|------|
| &lt; 0.30 | 公共安全秩序维持 | 传统海军护航有效，通行权为国际公共品 |
| 0.30-0.60 | 灰区萌芽 | 双边安排出现，但尚未制度化 |
| **0.60-0.85** | **灰区制度化** | **保险双轨制 + 五级分层 + 通行证体系运行** |
| &gt; 0.85 | 灰区秩序固化 | 公共安全彻底退场，碎片化的双边/私营/条件化安全成为常态 |

&gt; **L1 公开边界**：刻度含义公开。具体当前值与权重属于 L2 审计内容。

## 4. Gulf Diplomatic Stress (GDS)

度量海湾调停者（巴基斯坦等）的斡旋能力与经济脆弱性约束。

| GDS区间 | 状态 | 调停者功能 |
|---------|------|-----------|
| 0.00-0.30 | 调停失效 | 已无斡旋能力或意愿 |
| 0.30-0.50 | 严重受损 | 通道存在但效率极低 |
| 0.50-0.70 | 功能受限 | 可斡旋但受经济压力制约 |
| **0.70-0.85** | **基本正常** | **具备能力，但面临脆弱性约束** |
| 0.85-1.00 | 功能完整 | 无重大外部经济压力 |

&gt; **触发阈值**：GDS 从 0.80 骤降至 0.50 以下时，触发"调停通道不可逆关闭"预警。具体 ΔHEI 对应属于 L2 审计内容。

## 5. Verification Index (VXI)

外交成果的可验证性评级。

| 级别 | 判定标准 |
|------|----------|
| **A级** | 联合声明/公报 + 具体时间表 + 执行机制 + 监督框架 |
| **B级** | 联合声明但空泛 / 单方面声明但含可验证承诺 |
| **C级** | 无联合声明 / 仅有空泛修辞 / 无具体执行机制 |
| **D级** | 会谈破裂 / 公开相互指责 |

**操作含义**：
- **VXI A级**：妥协传导模型的输入条件满足，物理世界预期可随外交突破同步改善
- **VXI C级**：传导输入的初始条件未满足，Path Beta 激活，不基于会谈成果做出方向性押注

## 6. OSINT Fidelity & Auditing

### 6.1 验证标准
- **交叉验证**：每项主张至少交叉验证两个独立 OSINT 流（AIS、卫星影像、多语种官方声明）。
- **信源分级**：所有数据点标注信源路径与可靠性评级（可核实 / 方向确认 / 单一信源）。

### 6.2 脱敏政策
- 为保护 L3 订阅者的战略优势，公开文件中特定的时间-价格触发器已脱敏（[DATA REDACTED]）。
- 脱敏标记本身构成"情报局内参"质感，同时保护核心参数。

### 6.3 可审计性
- 所有逻辑通过 GitHub 时间戳留存，防止回溯偏差。
- 历史预判与事实验证记录存放于 `/Proof_of_Timestamp/` 目录。

### 6.4 认知天花板
- 国家间私下交易（非货币补偿、情报共享、秘密支付渠道、IRGC 独立征收体系）原则上超出 OSINT 验证能力。
- 当报告描述某国"通过外交交涉"获得通行便利时，不应被解读为"仅通过公开外交手段即可达成"，而应理解为"公开层面可观测到的部分外交互动，其完整交易结构不可知"。
- 这一认知局限是所有基于开源情报的地缘政治分析的固有属性，不可消除，只能标注。
