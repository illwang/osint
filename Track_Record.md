## 记录规范

| 字段 | 说明 |
| --- | --- |
| **日期 (UTC+8)** | 信号捕捉时间 |
| **异常信号** | OSINT 捕捉到的偏离共识的异常 |
| **逻辑判定** | 基于四层架构推导的结论 |
| **论证强度** | `确认性判断` / `方向性判断` / `推测性观察` / `分析性推断` |
| **信源路径** | 可复现的证据链入口（URL / 文档编号 / 模型输出标识） |
| **验证状态** | `✅ 已验证` / `⏳ 监测中` / `⏳ 信源待核` / `❌ 证伪` |
| **不可知维度** | 该结论依赖但超出公开信息验证能力的变量 |
| **证伪条件** | 何种公开信号出现即可推翻该判定 |

## 历史对账存证

本仓库 `/Proof_of_Timestamp/` 目录保留预判时间戳与事实链的交叉验证记录。每条记录包含：

- 预判原文截图或文本存档
